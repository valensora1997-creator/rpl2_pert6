## Deskripsi Project
Project ini dibuat untuk memenuhi tugas praktikum mata kuliah **RPL 2 (Rekayasa Perangkat Lunak 2)** pada pertemuan ke-6 dengan menerapkan konsep **AOP (Aspect Oriented Programming)**.

Aplikasi ini merupakan aplikasi pengolahan data **Mahasiswa** yang terdiri dari beberapa komponen utama:
- Controller
- Model
- Repository
- Service
- View

Project dikembangkan menggunakan:
- Java
- Maven
- Konsep MVC + AOP
